# Screenshots

Place the app screenshots in this folder so they appear in the main README on GitHub.

## Required files

| Filename | Screen |
|----------|--------|
| `splash.png` | Splash screen (logo) |
| `welcome.png` | Welcome / onboarding screen |
| `login.png` | Login screen |
| `dashboard.png` | Main dashboard |
| `camera.png` | Camera / package analysis screen |
| `history.png` | Analysis history |
| `settings.png` | Settings screen |
| `analysis-result.png` | Result screen (Intact/Damaged) |

## How to capture

1. Run the app: `npm start` → press `a` (Android) or `i` (iOS) or `w` (web).
2. Navigate to each screen and take a screenshot:
   - **Android emulator:** `Ctrl + S` or toolbar camera icon.
   - **iOS simulator:** `Cmd + S`.
   - **Physical device:** device screenshot (e.g. Power + Volume Down).
3. Save each image in this folder with the exact names above (PNG recommended).
