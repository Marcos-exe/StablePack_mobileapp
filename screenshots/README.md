# Screenshots

Screenshots are stored in **`assets/screenshots/`** and displayed in the main README.

|----------|--------|
| `Splash.png` | Splash screen |

| `LoginClient.png` | Login |

## Adding or updating screenshots

1. Run the app: `npm start` ‚Üí press `a` (Android), `i` (iOS), or `w` (web).
2. Capture each screen (emulator shortcut or device screenshot).
3. Save images in `assets/screenshots/` with the names above so the README links stay valid.
